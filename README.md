# viral-news-website
A responsive Viral News website built using HTML, Tailwind CSS, and Vue.js with categorized news cards and a virality analytics dashboard.
Viral News Website

A responsive news website dashboard built using HTML, Tailwind CSS, and Vue.js.
This project displays categorized news cards and includes an interactive Virality Analytics Dashboard for simulating story reach and engagement.

---

📌 Project Overview

The Viral News Website is a front-end web project designed to present news in a clean, modern, and interactive layout.
It includes:

- A news feed section with categorized news cards
- A market ticker bar
- A weather/environment information panel
- A Virality Analytics section to calculate story reach using adjustable sliders
- A responsive UI built with Tailwind CSS
- Dynamic rendering using Vue.js

Method 1: Open directly in browser

1. Download or clone the repository
2. Open the project folder
3. Double-click index.html

Method 2: Run using VS Code Live Server

1. Open the project folder in VS Code
2. Install the Live Server extension
3. Right-click on index.html
4. Click Open with Live Server

📊 Virality Formula

The analytics dashboard uses the following formula:

Score = ((Impressions / 25000) + (Shares / 200)) × Severity

This score is used to classify the story virality level into:

- Low Pulse
- Moderate
- High Momentum
- Viral Surge
- Hyper Impact
